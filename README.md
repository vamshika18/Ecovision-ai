# EcoVision AI ♻️
Smart Waste Classification & Recycling Optimization System

## 🚀 Overview
EcoVision AI is an intelligent waste classification system that uses YOLOv8 for real-time detection and categorization of waste into 22 classes.

## ✨ Features
- Real-time waste detection using webcam
- Snap & Freeze detection system
- Temporal smoothing for accuracy
- Contamination percentage analysis
- Streamlit interactive dashboard

## 🧠 Tech Stack
- Python
- YOLOv8
- TensorFlow
- Streamlit
- WebRTC
- Roboflow

## 📊 Performance
- Trained on 13,568 images (augmented to 31,936)
- Achieved 91.4% mAP50 accuracy
